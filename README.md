# Shopping List

## Requirements

-   [Node.js LTS](https://nodejs.org/en/download/)

## Setup

```shell
git clone git@github.com:pchalupa/shopping-list-frontend-assignment.git
cd shopping-list-frontend-assignment
npm install
```

## Development

```shell
npm run dev

# unit tests
npm run test

# static analysis
npm run lint
npm run lint:css
npm run ts:check
```

## Release

```shell
npm run release
```
